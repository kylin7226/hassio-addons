
# Home Assistant Add-on: RK LLaMA

## 关于

这是一个Home Assistant的RK LLaMA Add-on，它允许你在Home Assistant中运行RK LLaMA AI助手。

## 功能

- 提供RK LLaMA AI助手服务
- 通过Web界面访问（端口8080）

## 安装要求

- 仅支持 aarch64 架构设备

## 使用方法

1. 安装并启动Add-on后，等待服务完全启动
2. 通过以下地址访问服务：
   - `http://your-home-assistant:8080`

## 配置

此Add-on不需要额外配置，安装后即可使用。
